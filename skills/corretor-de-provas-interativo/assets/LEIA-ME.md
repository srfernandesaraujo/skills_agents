# 🎨 Pasta de Mídias e Recursos Visuais (/assets)

Esta pasta serve para armazenar mídias de apoio visual para a Skill **"Professor corretor de provas"**.

## 🎯 O que colocar aqui (Recomendado para esta Skill)?
* Imagens de exemplo de rubricas ou grades de correção, se o professor quiser um modelo visual.
* Prints/fotos de referência de provas manuscritas usadas para calibrar a leitura de letra manual (opcional).

## ⚙️ Como o Agente usa essa pasta?
Arquivos de imagem salvos aqui podem ser renderizados pelo Agente no chat usando formatação markdown padrão com caminhos relativos (ex: `![Modelo de rubrica](assets/modelo-rubrica.png)`). Esta skill normalmente não depende de arquivos fixos nesta pasta — as provas e gabaritos chegam como upload do professor durante a conversa.
